# Walmart Store Performance Optimization & Sales Forecasting

## Project Overview

This project analyzes Walmart retail sales data to identify revenue drivers, evaluate promotion effectiveness, measure holiday impact, and forecast future sales trends.

## Business Objective

The goal was to identify opportunities to improve store performance through data-driven decision making.

## Dataset

- 421,570 sales records
- 45 Walmart stores
- 81 departments
- Historical sales data (2010–2012)

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Prophet
- Google Colab

## Analysis Performed

### Data Quality Assessment
- Missing value analysis
- Data cleaning
- Date conversion

### Store Performance Analysis
- Top-performing stores
- Bottom-performing stores
- Revenue by store type
- Store size impact

### Department Performance Analysis
- Top revenue departments
- Department contribution analysis

### Holiday Impact Analysis
- Holiday vs non-holiday sales comparison
- Holiday sales lift calculation

### Promotion Effectiveness Analysis
- Promotion vs non-promotion sales
- Markdown impact evaluation

### Revenue Driver Analysis
- Correlation matrix
- Internal vs external factor evaluation

### Sales Forecasting
- Prophet forecasting model
- Trend analysis
- Seasonal pattern identification

## Key Findings

- Store 20 generated the highest revenue.
- Type A stores contributed the majority of sales.
- Department 92 generated the highest revenue.
- Promotions increased average sales by approximately 6.3%.
- Holidays increased sales by approximately 7.3%.
- Promotional activity had a greater impact on revenue than macroeconomic indicators.
- Forecasting indicates continued seasonal demand patterns.

## Business Recommendations

- Expand successful promotional strategies.
- Prioritize inventory allocation for high-performing departments.
- Increase staffing and inventory before holiday periods.
- Use forecasting models to support operational planning.
